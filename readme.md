## Dependancies
1. Make
2. Docker
3. Nodejs
4. Npm

## Running the app
### running from cli

`make run`

### running docker

`make up`

### stopping docker

`make down`

### push to docker hub

`make push`

https://hub.docker.com/r/dknathalage/calculator
